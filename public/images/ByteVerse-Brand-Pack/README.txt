# ByteVerse Brand Pack (PNG)

This pack contains minimal, symbol-first icons for the ByteVerse suite.

Variants per product:
- `transparent/` (white ink on transparent)
- `dark/` (black background)
- `light/` (soft cream background)

Sizes: 2048, 1024, 512 (square).

Products:
- ByteAI — core nucleus
- ByteVerse — connected V-lattice
- ByteNimbus — design cloud
- ByteTok — social bubbles
- ByteLab — experiment glyph
- ByteAnalytics — bars + line
- ByteSim — branch + loop
- ByteHub — library shelves
- ByteGenie — wand + path

Tips:
- Prefer **transparent** PNG for docs/sites with custom background.
- Use **512px** for favicons/app icons; **1024/2048** for decks and hero sections.
